deno安装
发布于 2019-01-30 / 更新于 2019-01-31 / deno /

deno 是新出的框架,本来想安装学习一下的,但是发现按照官方文档上的安装方法根本安装不了,应该是地址被墙了,折腾各种工具也没安装成功

官方的shell安装命令

curl -L https://deno.land/x/install/install.sh | sh
export PATH=$HOME/.deno/bin:$PATH

后来发现其实这个地址会跳转到github上的地址,所以完全可以用github上的地址替换原来的地址,命令如下

curl -L https://raw.githubusercontent.com/denoland/deno_install/master/install.sh | sh
export PATH=$HOME/.deno/bin:$PATH

速度有点慢,下载了十多分钟,总算是安装成功了,看截图
